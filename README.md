# Simple social news and discussion platform

## Technologies
* Node.js
* Express.js
* MongoDB
* Passport.js
* EJS
* Bootstrap 4

## Scope
* Here you can sign up, share a post or leave a comment.
* Posts and comments are editable by the author
* User must be signed in, to see content
